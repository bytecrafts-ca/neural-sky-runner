# Neural Sky Runner

Gesture-first endless runner for Meta Ray-Ban Display.

## Live demo

**https://bytecrafts-ca.github.io/neural-sky-runner/**

Paste that URL into the Meta AI app to test on glasses.

## Deploy

Every push to `main` rebuilds and publishes GitHub Pages automatically.

```bash
git add -A && git commit -m "your message" && git push
```

## Local dev

```bash
npm install
npm run dev
```

Client only: `npm run dev -w client` → http://localhost:5173
